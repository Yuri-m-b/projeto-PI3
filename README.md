# Projeto para PI3
Alunos: Conrado e Yuri  
Tema: Conjunto de tarefas para RPi3 no RTOS VxWorks.

## Etapa 1: Planejamento e Pesquisa

**Objetivo**: Compreender os requisitos do projeto e a implementação necessária.

    Tarefas:
        Pesquisar sobre a configuração do VxWorks em Raspberry Pi 3.
        Estudar a implementação de tarefas no RTOS VxWorks.
        Revisar como enviar e receber mensagens UDP em VxWorks.

**Entrega**: Documento de pesquisa com uma visão geral dos conceitos, definições e estratégias de implementação.
## Etapa 2: Configuração do Ambiente

**Objetivo**: Preparar os RPi3 para o desenvolvimento e testes.

    Tarefas:
        Instalar o VxWorks no Raspberry Pi 3.
        Configurar a rede para comunicação entre os RPi3.
        Testar a conectividade de rede entre os dispositivos.

**Entrega**: Relatório de configuração, incluindo capturas de tela e testes de conectividade.
## Etapa 3: Implementação das Tarefas

**Objetivo**: Criar as tarefas que enviarão e receberão mensagens.

    Tarefas:
        Desenvolver a tarefa de envio no RPi3 que envia mensagens.
        Desenvolver a tarefa de recebimento no RPi3 que recebe mensagens.
        Repetir o processo para as duas outras tarefas em cada RPi3.

**Entrega**: Código-fonte das tarefas e um manual de instruções para compilar e executar.
## Etapa 4: Implementação da Comunicação UDP

**Objetivo**: Configurar a comunicação UDP entre as tarefas.

    Tarefas:
        Implementar a lógica de envio de mensagens UDP na tarefa de envio.
        Implementar a lógica de recebimento de mensagens UDP na tarefa de recebimento.
        Testar a troca de mensagens entre as duas RPi3.

**Entrega**: Código-fonte da implementação da comunicação UDP e um relatório de testes com resultados.
## Etapa 5: Registro de Logs

**Objetivo**: Criar funcionalidades de logging para monitorar a troca de mensagens.

    Tarefas:
        Adicionar logging para a tarefa de envio, registrando quantas mensagens foram enviadas.
        Adicionar logging para a tarefa de recebimento, registrando quantas mensagens foram recebidas.
        Testar e garantir que os logs sejam gerados corretamente.

**Entrega**: Código-fonte atualizado com as funcionalidades de logging e um relatório de verificação dos logs.
## Etapa 6: Testes e Validação

**Objetivo**: Realizar testes finais para garantir que tudo funcione como esperado.

    Tarefas:
        Testar a comunicação entre todas as tarefas nos RPi3.
        Avaliar o desempenho e a confiabilidade da troca de mensagens.
        Corrigir quaisquer bugs ou problemas encontrados durante os testes.

**Entrega**: Relatório final com os resultados dos testes, correções feitas e avaliação do projeto.
## Etapa 7: Documentação e Apresentação

**Objetivo**: Preparar a documentação do projeto e a apresentação final.

    Tarefas:
        Compilar toda a documentação do projeto, incluindo manuais e relatórios.
        Criar uma apresentação para compartilhar os resultados e a experiência do projeto.

Entrega: Documento completo do projeto e apresentação em slides.
